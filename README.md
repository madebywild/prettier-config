# @madebywild/prettier-config

```bash
npm i @madebywild/prettier-config prettier --save-dev
```

Then, edit your projects `package.json`:

```json
{
  "prettier": "@madebywild/prettier-config"
}
```

How to bump version:

Install `np` from https://github.com/sindresorhus/np

```bash
npm run release
```
